# 🧾 01 — Resume Checklist

## Data Science Resume Must-Haves

- clear headline
- skills grouped by category
- project bullets with metrics
- tools and libraries
- links to GitHub/portfolio
- concise education/certifications

## Strong Project Bullet Formula

```text
Built [model/analysis] to solve [problem], using [tools], achieving [metric/result], resulting in [business value].
```

Example:

```text
Built a churn prediction pipeline using Pandas and scikit-learn, improving recall for churners to 82% and identifying high-risk customer segments for retention targeting.
```

## Avoid

- listing tools without proof
- vague bullets like "worked on data"
- screenshots without code
- no metrics
